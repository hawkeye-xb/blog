Terminal-Hub 是一个终端风格的 Hugo 主题，专为独立开发者设计。用它展示你的作品、文章和想法。

## 特性

- **终端美学** — 暗色主题 + 等宽字体 + CLI 交互风格，三套配色方案（green / amber / cyber）
- **项目展示** — 内置项目卡片组件，支持状态标识、技术栈标签、外部链接
- **全文搜索** — 集成 Pagefind，静态站点也能有快速搜索
- **轻量极速** — 纯 Vanilla JS + CSS，无框架依赖，JS < 30KB、CSS < 50KB

## 技术栈

- Hugo 静态生成（要求 v0.120.0+）
- 纯 JavaScript 交互，零框架依赖
- Pagefind 全文检索
- Cloudflare Pages 部署
